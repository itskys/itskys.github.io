#### 1. Install gcc
```
	gcc -v
	yum install gcc
```
   

#### 2. install rvm
```
	cd /home/
	curl -L get.rvm.io | bash -s stable
	vi .bashrc
	  source /etc/profile.d/rvm.sh
	rvm -v
```
   

#### 3. install ruby
```
	rvm install 
	rvm list
	rvm use
	ruby -v
	gem -v
	gem update --system
	gem -v
```
   

#### 4. install jekyll
```
	gem install ruby-devel
	gem install jekyll
	jekyll -v
```
   

#### 5. install others
```
	gem install RedCloth //
	gem install rdiscount //
	gem install kramdown //从2016.03.01起，github只支持这个引擎
	jekyll --rdiscount
	jekyll --kramdown
	markdown: kramdown
	yum install python-pyments
```
   

#### 6. Run jekyll
```
	jekyll build
	jekyll --server
```
